安装laravel版本要根据机器上的php版本选择

### #步骤

1. composer install
2. php artisan sail:install // 初始化 sail
3. php artisan sail:publish





Linux命令

> 设置别名
>
> alias sail='bash vendor/bin/sail'
>
> asias up
